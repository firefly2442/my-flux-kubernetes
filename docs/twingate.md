# Twingate

Twingate is a modern, cloud-based remote access solution that replaces traditional Virtual Private Networks (VPNs).

## Setup

The secret contains base64 encoded values of both the `TWINGATE_ACCESS_TOKEN` and `TWINGATE_REFRESH_TOKEN`.
