# Karakeep

A self-hostable bookmark-everything app (links, notes and images) with AI-based automatic tagging and full text search

## LLM Integration

Karakeep -> LiteLLM -> Gemini

A virtual key is setup in LiteLLM as the API key that is setup in Karakeep in
the environment variable `OPENAI_API_KEY`.  See
[details in the Karakeep docs](https://docs.karakeep.app/configuration/different-ai-providers/#option-1-openai-compatible-api-recommended).
