# Rancher

Rancher Dashboard has a nice UI for managing your Kubernetes cluster.
However, it's very sensitive to k3s version updates.  It seems to lag behind
their latest version fairly often.

Given all this and the complexity of the Rancher ecosystem, it's commented
out and not being used.
